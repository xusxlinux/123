## 搭建es集群
|软件|版本|
|:-|:-|
|jdk|jdk-8u191-linux-x64.tar.gz|
|elastic|elasticsearch-7.17.7-x86_64.rpm|

#### 准备步骤
``` shell
# 所有的es节点都需要创建目录
mkdir -pv /data/{es-data,es-logs,es-plugins}

```

####
``` shell
# 验证集群是否搭建成功
curl http://10.4.7.11:9200/_cat/nodes
curl http://10.4.7.11:9200/_cluster/health?pretty
```
